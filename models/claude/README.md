# Claude 4.5 Sonnet Computer Use for Heatseeker

This module implements Anthropic's Claude 4.5 Sonnet computer use feature to autonomously play the Heatseeker game.

## Setup

This project uses `uv` for package management. Set up the environment:

```bash
uv venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate
uv pip install -e .
```

## Running

See the main documentation at `/docs/computer_use/claude-4_5-api.md` for instructions on running the computer use implementation.

## Testing

Run tests with:

```bash
pytest
```
